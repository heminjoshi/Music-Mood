# Music-Mood
## Usage
Install requirements
```bash
pip install -r requirements.txt
```

Run Server
```bash
python app.py
```

## Classifier Information - V1
* Model : **Support Vector Machine**
  * Training F1 : 0.997
  * Validation F1 : 0.712

* Method : TFIDF - Vectorizer 
  * Stopwords : English
  * N-Gram Range : (1,5)
